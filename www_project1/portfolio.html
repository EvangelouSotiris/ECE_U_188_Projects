Network Intrusion Detection System using Machine Learning Techniques

For the course of Machine Learning, we created a system that uses mainly Multilayer Perceptron (compared to SVM and RandomForest) using network packet metadata to decide whether there is an ongoing attack on the system, with a low false positive rate. With less accuracy, the system might also predict the type of attack happening (DoS, Backdoor etc.). The project was considered by the professor of the course as top 3.

Took data from Aminer.com regarding citations of academic papers over the years, processed them and created a Long Short-Term Memory neural network that trains on these data and predicts the number of citations a paper can get next year, and after 5 years given a citation timeseries of the former years.
